For input: Give single sudoku puzzle 9x9 in space separated format in "input.txt" file
Note: Be careful for input file. There should only 9 lines(one for each row without empty line).

For output: Output will save in output.txt file after finding solution.

How to run code?
1. Run below python command in CMD to run backtrack algorithm
Command: python SudokuSolver_Backtracking.py

2. Run below python command in CMD to run minimum conflict algorithm
Command: python SudokuSolver_minconflict.py

All testcases with solution are available in "Testcases and their solutions.txt"
